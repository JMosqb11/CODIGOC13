INICIO CARRERAS

FRONTEND

Es la parte visible, la que muestra el diseño, los contenidos y la que permite a los visitantes navegar por las diferentes páginas mientras lo deseen. Frontend se usan otros términos interfaz de usuario.

BACKEND

Este sistema recoge información de los usuarios u otros sistemas de tratamiento de datos en la compañía. Es el encargado de gestionar la información que proporciona el usuario recogida por el sitio web.

FULL STACK

Un Full Stack es la persona responsable de la creación y el mantenimiento de una aplicación web. Por este emotivo, es importante que el Full Stack tenga conocimientos de desarrollo Front-End y Back-End.
